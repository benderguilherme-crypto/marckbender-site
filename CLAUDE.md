# Site Marck Bender — Vision IA

## Contexte
Site personnel de Marck Bender, consultant IA basé sur le Blueprint v4.0.
Positionnement : "L'IA sans vision, c'est de l'exécution aveugle."
Marché : francophone (France, Belgique, Suisse).

## Stack technique
- HTML/CSS/JS pur dans un seul fichier index.html
- Pas de framework, pas de build
- Photos en base64 directement dans le HTML
- Police : Inter (Google Fonts) + Instrument Serif (italiques)

## Palette
- Fond clair : #f5f5f7
- Marine : #0a1628
- Bleu métal poli (gradient) : #a8d4f5 → #5a9fdb → #2563b8 → #1a4d8c
- Texte muted : #5a6478

## Sections du site
1. Hero avec terminal Claude Code animé
2. Trust bar (HEC, Telos, etc.)
3. Big Idea (citation fondatrice)
4. 4 Solutions (Vision, Claude Code, Architecture, Back-End)
5. Stats (11+ ans, 500+ entrepreneurs, HEC)
6. Bio
7. Philosophie (2 piliers)
8. Offres (escalier 27€ → 3000€)
9. YouTube (dernière vidéo auto via RSS)
10. Contact (formulaire qualifié par ICP)
11. Footer

## Règles de travail
- Toujours préserver la palette bleu métal poli avec ses ombres internes
- Tout reste dans un seul fichier index.html (pas de séparation CSS/JS)
- Ne jamais casser le script de récupération auto YouTube
- Le ton est direct, expert, sans bullshit (cf. Blueprint v4.0)
